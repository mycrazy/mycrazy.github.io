静态化处理
通过一条命令，把所有的文章都做静态化处理，就是生成对应的html, javascript, css，使得所有的文章都是由静态文件组成的。
hexo generate
在本地目录下，会生成一个public的目录，里面包括了所有静态化的文件。

部署命令
hexo deploy
如果部署完全没有反应，在部署之前可能还需要执行如下命令，安装git插件
npm install hexo-deployer-git –save
执行成功之后，重新使用hexo deploy命令，部署到github上。

然后在浏览器中打开http//username.github.io你就可以看到你的blog了。
当然如果你感觉自带的主题不合你的口味，可以自定义主题或者在这个https://hexo.io/themes/
网站下载主题，然后安装即可。
